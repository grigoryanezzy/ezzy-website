# Ezzy — Химчистка мебели в Ташкенте

🛋️ Профессиональная химчистка мягкой мебели, диванов и матрасов в Ташкенте.

## 🔗 Ссылки

- **Live сайт:** https://ezzy-website-v2.vercel.app
- **GitHub:** https://github.com/grigoryanezzy/ezzy-website

## ✨ Возможности

- **Главная страница** с Hero-секцией, услугами и статистикой
- **6-шаговый квиз** для расчёта стоимости:
  1. Что чистим (диван, кресло, матрас, ковёр и т.д.)
  2. Размер (маленький / средний / большой)
  3. Типы загрязнений (вино, моча, жир, кровь, кофе, грязь)
  4. Где находится мебель
  5. Когда удобно (сегодня / завтра / эта неделя)
  6. Контактные данные
- **Скидка 10%** на первый заказ
- **Форма заказа** с именем и номером телефона
- **Адаптивный (мобильный) дизайн**
- **Отзывы клиентов**

## 🛠️ Технологии

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**
- **Lucide React** (иконки)
- **PT Sans** (Google Fonts)
